
# Socket io test

Project created to test my skills in creating a simple chat with socket io


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Functionalities

- Real-time messages
- Preview em tempo real
- Viewing Users Online
